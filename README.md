## General info
This project is simple blog where user can sign up to add posts/thoughts.
	
## Technologies
Project is created with:
* React version: 17.0.1
* Firebase: 8.2.1
* Bootstrap: 4.5.3


## Setup
To run this project, install it locally using npm:
In the project directory
```
$ npm install
$ npm start
```
